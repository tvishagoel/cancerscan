⚙️ How it works
1
Segment cells — Gaussian smoothing → Otsu threshold → Watershed (same algorithms as CellProfiler)
2
Measure 12 AreaShape features per cell: area, eccentricity, solidity, perimeter, compactness and more
3
Within-image outlier detection — finds cells that look different from all OTHER cells in the same image using IsolationForest + Tukey's fence rule
4
Flag suspects — only cells that are extreme outliers (Tukey fence: Q1 − 2×IQR) get flagged. Borderline cells stay green.
✅ Why this is reliable: Healthy tissue = all cells look similar to each other → no outliers → all green. Cancer tissue = cancer cells look different from surrounding healthy cells → those specific cells get flagged red. No external reference needed — no domain mismatch.
